# Taiwan Bicycle Industry Statistics

Static dashboard for Taiwan bicycle industry trade statistics.

## Deploy With GitHub Pages

Use this folder as the repository root.

1. Create an empty GitHub repository.
2. Push every file in this folder to the repository `main` branch.
3. In GitHub, open `Settings` > `Pages`.
4. Set `Source` to `GitHub Actions`.
5. The included workflow will publish the site on every push to `main`.
